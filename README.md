# MAP-Innovative_Assignment

## Install Dependencies

```
$ pip3 install flask
$ pip3 install flask-restful
$ npm install
```

## Run API Gateway

Terminal 1
```
$ node gateway.js
```

Terminal 2
```
$ node take.js
```

Terminal 3
```
$ node modify.js
```

Terminal 4
```
$ node view.js
```

Terminal 5
```
$ python3 home_app.py
```
