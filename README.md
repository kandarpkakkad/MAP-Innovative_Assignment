# MAP-Innovative_Assignment

## Install Dependencies

```
$ pip3 install flask
$ pip3 install flask-restful
$ pip3 install gunicorn
$ npm install
```

## Run API Gateway

Terminal 1
```
$ node gateway.js
```

Terminal 2
```
$ python3 take.py
```

Terminal 3
```
$ python3 modify.py
```

Terminal 4
```
$ python3 view.py
```

Terminal 5
```
$ python3 home_app.py
```
